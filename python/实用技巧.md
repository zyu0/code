# 实用技巧

1. 在注释加 # 通过 ctrl + / 

2. 给大量注释增加缩进可全选中 Tap ，返回用 Shift + Tap

3. if () or () or ():

	- 用于复杂逻辑时的技巧

	- 若太长 if (() or () or ())  使or换行

 4. 要先写完好识别的逻辑，再考虑其他逻辑

 5. 实用单步调试工具

	- F8

	- F7

	- F9

 6. Pycharm技巧

	- 在（ ）内部的过长字符串进行换行会自动实现分割字符串

	- shift +F6 重命名

7. 快捷键 ctrl+ . 折叠选中的部分

8. ==打印一段代码返回值的快速写法==

   ```python
   foo.prop().print  
   # 后面加 .print则IDE自动转成以下
   print(foo.prop())
   ```
