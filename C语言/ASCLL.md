内存中数据的表现形式

一套规范，把所有字符进行标号

```c
void main()
{
	int a = 1;
	char b = "1";
	printf("%d,%d\n", sizeof(a), sizeof(b));
	putchar(105);
	putchar(108);
	putchar(111);
	putchar(118);
	putchar(101);
	putchar(121);
	putchar(111);
	putchar(117);

	system("pause");

}
```
